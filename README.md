# AgriMitra 🌱🤖

## AI Multi-Agent Smart Farming Assistant

AgriMitra is an AI-powered agriculture platform designed to help farmers solve real-world farming problems using intelligent AI agents. It provides instant support for crop disease detection, weather guidance, waste management, market insights, and smart farming recommendations.

## 🚀 Live Demo

https://praveen-kumar0333.github.io/AgriMitra/

---

## 📌 Problem Statement

Farmers often face challenges such as:

* Crop disease identification delays
* Uncertain weather conditions
* Poor waste management
* Lack of market price awareness
* Limited access to expert guidance
* Difficulty finding nearby agricultural medicine shops

Traditional solutions are slow, costly, and not always accessible.

---

## 💡 Our Solution

AgriMitra uses AI-powered specialized agents to provide fast, simple, and intelligent farming support anytime, anywhere.

Users can choose different AI agents based on their needs.

---

## 🤖 AI Agents Included

### 1. Crop Doctor Agent

* Detects crop disease using uploaded leaf images or text symptoms
* Suggests treatment methods
* Gives prevention tips

### 2. Weather Advisor Agent

* Current weather updates
* Irrigation advice
* Spray warnings
* Farming recommendations

### 3. Waste to Wealth Agent

* Crop waste reuse ideas
* Compost suggestions
* Biofuel and income opportunities

### 4. Market Price Agent

* Crop selling guidance
* Price trends
* Better decision support

### 5. Government Scheme Agent

* Farmer subsidy schemes
* Insurance and loan support

### 6. Emergency Help Agent

* Pest attack guidance
* Flood / drought precautions

## ⚙️ How It Works

1. User selects an AI agent
2. Uploads plant image or enters symptoms
3. AI analyzes the issue
4. Runs the correct agent workflow
5. Provides treatment and recommendations
6. Suggests nearby shops if medicine is needed
